Privacy Policy

Last Updated: April 2026  ·  How HaxEm Studio handles your information.

1

Information We Collect

Most HaxEm Studio plugins do not collect personal data from end users. Depending on the product, the following technical data may be processed locally:

  • Minecraft player UUID
  • Player username
  • Server IP address (for whitelist/authentication features)
  • Plugin configuration data
  • Server environment information (for debugging purposes)

This information is used solely for plugin functionality and is not transmitted externally by default.

2

Data Storage

Data may be stored locally on your server using SQLite databases, MySQL databases, or plain configuration files. HaxEm Studio does not automatically collect or store this data on external servers. All data remains under the full control of the server owner.

3

Third-Party Integrations

Some plugins may integrate with optional third-party services such as PlaceholderAPI, EssentialsX, or external databases (MySQL/MariaDB). These integrations are fully optional and controlled by the server administrator. HaxEm Studio is not responsible for the data practices of third-party services.

4

Analytics and Metrics

Some plugins may include anonymous usage metrics via bStats, such as plugin version, server software type, or basic environment data. This helps us improve stability and compatibility. No personally identifiable information (PII) is collected. You may disable bStats globally via your server's bStats configuration file.

5

Data Control

Server owners retain full control over all data generated by HaxEm Studio plugins. You may delete databases, disable optional metrics, and modify or remove configuration files at any time without restriction.

6

Security

HaxEm Studio takes reasonable precautions to ensure that plugins do not expose sensitive information or introduce security vulnerabilities. However, server owners are independently responsible for maintaining a secure server environment.

7

Changes to This Policy

This Privacy Policy may be updated at any time. Changes will be reflected with an updated "Last Updated" date. Significant changes will be communicated through our official Discord server where possible.

8

Contact

If you have questions or concerns regarding this Privacy Policy or your data, please contact HaxEm Studio through our official support channels on Discord.
